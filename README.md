# Litgrid – Pirkimų Trukmių Skaičiuoklė

Interaktyvus įrankis pirkimo proceso trukmės skaičiavimui darbo dienomis,
įvertinant Lietuvos valstybines šventes (2026–2028 m.).

## Apie įrankį

Pirkimo trukmė susideda iš dviejų etapų:
- **Pasirengimo trukmė** – techninė specifikacija ir pirkimo dokumentai
- **Vykdymo trukmė** – procedūrų vykdymas iki sutarties pasirašymo

Skaičiavimai grindžiami WORKDAY formule su pataisos koeficientu
(pradžios diena įskaičiuojama į trukmę).

## Skaičiuoklių tipai

| # | Spalva | Žinoma | Apskaičiuojama |
|---|--------|--------|----------------|
| 1 | 🟠 Oranžinė | Inicijavimo data | Pirkimo pradžia + sutarties data |
| 2 | 🟢 Žalia | Planuojama sutarties data | Kada pradėti pasirengimą |
| 3 | 🟡 Geltona | Pirkimo pradžios data | Sutarties sudarymo data |

## Palaikomi pirkimo būdai (pavyzdžiai)

- T-SD · Tarptautinis pirkimas – skelbiamos derybos (25+110 d.d.)
- (kiti būdai pagal PSĮ ir VPT taisykles)

## Techninė informacija

- Skaičiavimai atitinka Litgrid naudojamas WORKDAY formules
- Įvertinamos Lietuvos valstybinės šventės 2026–2028 m.
- Savaitgaliai automatiškai praleidžiami
